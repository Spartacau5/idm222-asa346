Nightscape

A prototype I made for my IDM110 Final Presentation. Nightscape is a social platform for content creators to genuinely engage and critique each others' work. It's Unique Selling Point (USP) is that if the user doesn't actively use the app, it drops the users' rating. This leads to lesser reach of posts from the user which in turn leads to less engagement for the user itself. Thus, the user is promoted to use the app to it's fullest.
